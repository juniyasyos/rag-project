# AI Agent Usage

Panduan utama untuk AI Agent saat memproses repository ini:

1. **Jika task tentang project ini**, pakai `rag-project` sebagai peta awal.
2. **Jika task tentang RAG/parser/chunks/graph/query/ingest**, mulai dari `rag-project/`.
3. **Jangan baca seluruh repo** kalau RAG/docs cukup.
4. **Kalau RAG kurang jelas/gagal**, baru baca source file relevan.
5. **Source code tetap source of truth**.

Selalu rujuk ke dokumen ini dan `rag-project/` untuk memahami arsitektur project.
